# PROJETOS_CPP
📘 **Repositório de Exercícios em C++**

Este repositório reúne exercícios de programação em C++, desenvolvidos com base nos seguintes livros:

Algoritmos – Lógica para Desenvolvimento de Programação Imperativa de Computadores  
Autores: José Augusto N. G. Manzano e Jayr Figueredo de Oliveira (30ª edição)

C++23 para Windows - Guia de Introdução para Iniciantes
Autor: José Augusto N. G. Manzano (1ª edição)

O objetivo é praticar e consolidar conceitos fundamentais de programação imperativa e da linguagem C++ moderna.

📂 **Estrutura dos Exercícios**

Os exercícios estão organizados por tópicos de aprendizado:

Programação com Sequência

Entrada e saída de dados

Operações aritméticas básicas

Exercícios introdutórios

Programação com Decisão

Estruturas condicionais (if, else, switch)

Exercícios de lógica com múltiplos caminhos

Programação com Laços

Estruturas de repetição (for, while, do-while)

Exercícios de contagem, acumulação e repetição controlada

Programação com Matrizes

Vetores (arrays unidimensionais)

Matrizes bidimensionais

Exercícios de manipulação de coleções de dados

Programação com Sub-rotinas

Funções definidas pelo usuário

Passagem de parâmetros por valor e por referência

Modularização de programas

Programação com Estruturas de Dados Heterogêneas

struct e class

Exercícios com registros e objetos simples

Introdução à organização de dados complexos

🚀 **Objetivos**

Fixar conceitos de lógica de programação imperativa.

Aplicar recursos da linguagem C++23 em exemplos práticos.

Desenvolver habilidades de resolução de problemas com código limpo e estruturado.

Criar uma base sólida para estudos avançados em programação e ciência da computação.
